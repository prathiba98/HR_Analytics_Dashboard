# HR Analytics Dashboard
## Dashboard link - https://hranalyticsdashboard-5h4cr5n8f9ncq2dyjs5dxp.streamlit.app/
## Overview
This project is an interactive HR analytics dashboard built using Streamlit.
It analyzes employee attrition patterns to help HR teams understand why
employees leave and identify high-risk groups.

The dashboard is based on the IBM HR Analytics Employee Attrition dataset
and focuses on key factors such as department, overtime, income, age, and
job satisfaction.

---

## Key Features
- Dynamic KPI metrics:
  - Total Employees
  - Active Employees
  - Attrition Count
  - Attrition Rate
- Interactive sidebar filters:
  - Department
  - Gender
  - Job Role
- Attrition analysis by:
  - Department
  - Overtime
  - Monthly Income
  - Age
  - Job Satisfaction
- Business-focused insights and recommendations
